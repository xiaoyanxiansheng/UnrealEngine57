# 4.3 阴影系统 - 速查卡

## 🎯 核心结论

1. **CSM**：Cascaded Shadow Maps，平行光4级级联，近高远低分辨率
2. **VSM**：Virtual Shadow Maps，16K虚拟空间 + 128×128物理Page + 80-95%缓存命中
3. **Contact Shadows**：屏幕空间Ray March，补全近距离自阴影细节
4. **Shadow Bias**：Constant + Slope自适应，平衡Acne与Peter Panning
5. **VSM是UE5.7重要路径**，缓存后通常能降低重复阴影渲染成本，具体收益需项目实测

## 📊 阴影技术对比

| 技术 | 适用 | GPU耗时 | 特点 |
|------|------|---------|------|
| CSM | 平行光 | 2-3ms | 经典方案，全平台兼容 |
| VSM | 所有光源 | 0.5-2ms | 虚拟化+缓存，推荐方案 |
| Contact Shadow | 补充 | 0.3-0.5ms | 屏幕空间，近距离细节 |
| Ray Traced | 高端 | 8-15ms | 物理正确半影 |
