# 4.2 材质系统 - 速查卡

## 🎯 核心结论

1. **Material Expression**：节点图编译为HLSL
2. **Shader Permutation**：2^N变体组合
3. **Substrate**：UE5新材质模型，更物理
4. **编译期哈希**：FName快速查找
5. **表达式模板**：编译期优化AST

## 📊 材质编译流程

```
Material Editor（节点图）
    ↓
MaterialCompiler（AST）
    ↓
HLSL代码生成
    ↓
Shader Permutation（变体组合）
    ↓
FShaderType注册
    ↓
运行时加载
```

## 🔗 核心类索引

| 类 | 职责 |
|----|------|
| `UMaterial` | 材质资产 |

| 函数 | 作用 |
