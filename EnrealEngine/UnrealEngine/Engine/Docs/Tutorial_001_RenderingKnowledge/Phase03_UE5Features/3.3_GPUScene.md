# 3.3 GPUScene - 速查卡

## 🎯 核心结论

1. **GPU Driven**：场景数据存GPU Buffer，Shader直接读取
2. **TSparseArray**：稳定索引，O(1)增删
3. **Upload Buffer**：Ring Buffer异步上传
4. **Instance Culling**：GPU剔除 + Indirect Draw
5. **Tiled SoA**：混合布局，平衡访问效率

## 📊 GPUScene数据流

```
CPU: UPrimitiveComponent
    ↓ 同步
CPU: FPrimitiveSceneProxy
    ↓ Upload Buffer
GPU: GPUScene Buffer
    ↓ Compute Shader读取
GPU: Culling / Rendering
```

## 🔗 核心类索引

| 类 | 职责 |
|----|------|
| `FGPUScene` | GPU场景数据管理 |
| `FPrimitiveSceneShaderData` | Primitive GPU数据 |
| `FRDGAsyncScatterUploadBuffer` | 异步上传 |

## 🔗 核心函数索引

| 函数 | 作用 |
|------|------|
| `UploadDynamicPrimitiveShaderData()` | 上传数据 |
| `DispatchCulling()` | GPU剔除 |
| `DrawIndexedIndirect()` | 间接绘制 |
