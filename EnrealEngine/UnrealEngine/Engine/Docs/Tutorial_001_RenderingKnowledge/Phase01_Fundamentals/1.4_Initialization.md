# 1.4 初始化流程 - 速查卡

## 🎯 核心结论

1. **启动链**：main() → GuardedMain() → FEngineLoop::PreInit() → Init() → Tick()
2. **最大瓶颈**：Shader加载（33%），DDC缓存是关键优化点
3. **RHI初始化**：PlatformCreateDynamicRHI() 创建平台RHI（D3D12/Vulkan/Metal）
4. **反射系统**：UHT编译期生成反射代码，运行时通过StaticClass()访问
5. **模块加载**：FModuleManager 按依赖图拓扑排序加载

## 📊 启动时序（典型PC）

```
0ms      main()
50ms     GuardedMain()
100ms    FEngineLoop::PreInit()
         ├─ [100-150ms] 平台层初始化
         ├─ [150-200ms] TaskGraph初始化
         ├─ [200-250ms] 模块加载
         ├─ [250-550ms] ★ RHI初始化 (300ms)
         ├─ [550-1550ms] ★ Shader加载 (1000ms) ← 最大瓶颈
         └─ [1550-2000ms] 默认材质初始化
2000ms   FEngineLoop::Init()
         ├─ UGameEngine::Init()
         └─ LoadMap() → FScene::FScene()
3000ms   FEngineLoop::Tick() ← 首帧渲染
```

## 🔗 核心函数索引

| 函数 | 文件 | 作用 |
|------|------|------|
| `FEngineLoop::PreInit()` | LaunchEngineLoop.cpp | 核心初始化入口 |
| `PlatformCreateDynamicRHI()` | DynamicRHI.cpp | 创建平台RHI |
| `FShaderCompilingManager` | ShaderCompiler.cpp | Shader编译管理 |
| `FScene::FScene()` | Scene.cpp | 创建渲染场景 |
| `UClass::StaticClass()` | Class.cpp | 获取类反射信息 |
