# 1.1 渲染循环 - 速查卡

> **本章聚焦**：渲染调用链（从FEngineLoop到GPU）、核心源码解读、RenderDoc调试
> 
> **线程同步机制**（ENQUEUE原理、Fence、TaskGraph）→ 详见 [1.2 ThreadModel](./1.2_ThreadModel.md)

## 🎯 核心结论

1. **调用链入口**：`FEngineLoop::Tick()` → `GEngine->Tick()` → `Viewport->Draw()` → `FSceneRenderer::Render()`
2. **渲染器入口**：`FDeferredShadingSceneRenderer::Render()` 是延迟渲染帧入口
3. **渲染阶段**：InitViews → PrePass → BasePass → Lighting → Translucency → PostProcessing
4. **场景容器**：FScene 存储渲染数据，FSceneView 存储视图数据
5. **三线程流水线**：GT处理帧N，RT处理帧N-1，GPU执行帧N-2

## 📊 渲染调用链

```
GameThread                  RenderThread                RHIThread
    │                            │                          │
    │ FEngineLoop::Tick()        │                          │
    │  └─ GEngine->Tick()        │                          │
    │      └─ RedrawViewports()  │                          │
    │          └─ Viewport->Draw()│                         │
    │              │             │                          │
    │              └─ ENQUEUE ──>│                          │
    │                            │                          │
    │                  FSceneRenderer::Render()             │
    │                            │                          │
    │                  ├─ InitViews (可见性剔除)            │
    │                  ├─ PrePass (Depth Only)             │
    │                  ├─ BasePass (GBuffer填充)           │
    │                  ├─ Lighting (延迟光照)              │
    │                  ├─ Translucency (透明物体)          │
    │                  └─ PostProcessing (后处理)          │
    │                            │                          │
    │                            │ RHICommandList           │
    │                            ├─────────────────────────>│
    │                            │                          │
    │                            │                    GPU Submit
```

## 🔗 核心函数索引

| 函数 | 文件 | 作用 |
|------|------|------|
| `FEngineLoop::Tick()` | `LaunchEngineLoop.cpp`，搜索 `FEngineLoop::Tick` | 每帧入口 |
| `UGameEngine::Tick()` | `GameEngine.cpp`，搜索 `UGameEngine::Tick` | 游戏引擎Tick |
| `FViewport::Draw()` | `UnrealClient.cpp`，搜索 `FViewport::Draw` | 触发渲染 |
| `FDeferredShadingSceneRenderer::Render()` | `DeferredShadingRenderer.cpp`，搜索 `FDeferredShadingSceneRenderer::Render` | 渲染帧入口 |
| `InitViews()` | SceneVisibility.cpp | 可见性计算 |
| `RenderBasePass()` | BasePassRendering.cpp | GBuffer填充 |
