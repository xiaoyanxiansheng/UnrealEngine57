# 2.4 光照系统 - 速查卡

## 🎯 核心结论

1. **Deferred Lighting**：从GBuffer读取材质，逐光源计算
2. **光源类型**：Directional、Point、Spot、Rect
3. **Tiled/Clustered**：光源分块剔除优化
4. **IES Profile**：真实光源分布
5. **SIMD优化**：FVector4批量计算

## 📊 光照计算流程

```
GBuffer → 逐光源循环 → 累加到SceneColor
    │
    ├─ Directional: 全屏Quad
    ├─ Point/Spot: 光源体积
    └─ Clustered: 分块索引
```

## 🔗 核心函数索引

| 函数 | 文件 | 作用 |
|------|------|------|
| `RenderLights()` | LightRendering.cpp | 光照主入口 |
