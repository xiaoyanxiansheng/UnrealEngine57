# 2.3 GBuffer - 速查卡

## 🎯 核心结论

1. **5个RT**：GBufferA(Normal) + B(Metal/Rough) + C(BaseColor) + D(Custom) + E(Shadow)
2. **法线编码**：八面体编码（Octahedral），2通道存3D法线
3. **ShadingModelID**：4位，支持16种着色模型
4. **Substrate模式**：新材质系统，布局不同

## 📊 GBuffer布局

| RT | 格式 | 内容 |
|----|------|------|
| GBufferA | R8G8B8A8 | RGB:法线(八面体) A:PerObject |
| GBufferB | R8G8B8A8 | R:Metallic G:Specular B:Roughness A:ShadingModel |
| GBufferC | R8G8B8A8 | RGB:BaseColor A:AO |
| GBufferD | R8G8B8A8 | CustomData(SSS/Cloth/Eye等) |
| GBufferE | R8G8B8A8 | PrecomputedShadow |
| Velocity | R16G16F | 屏幕空间运动矢量 |

## 🔗 核心函数索引

| 函数 | 文件 | 作用 |
|------|------|------|
| `EncodeGBuffer()` | `DeferredShadingCommon.ush`，搜索 `void EncodeGBuffer` | 编码 |
| `DecodeGBuffer()` | DeferredShadingCommon.ush | 解码 |
| `OctahedronEncode()` | Common.ush | 法线编码 |

## 🆚 Unity HDRP对比

| 维度 | UE5 | Unity HDRP |
|------|-----|------------|
| RT数量 | 5+Velocity | 4+Velocity |
| 法线编码 | 八面体 | 八面体 |
| 着色模型 | 16种 | 固定几种 |
