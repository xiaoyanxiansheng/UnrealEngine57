# 2.2 BasePass - 速查卡

## 🎯 核心结论

1. **职责**：输出GBuffer，包含材质属性
2. **MeshPassProcessor**：FBasePassMeshProcessor处理网格
3. **Shader Permutation**：根据材质特性选择变体
4. **ParallelFor**：多线程生成MeshDrawCommand
5. **MeshDrawCommand**：缓存的绘制命令

## 📊 BasePass流程

```
FMeshPassProcessor::AddMeshBatch()
    ↓
生成 FMeshDrawCommand
    ↓
缓存到 FMeshDrawCommandStorage
    ↓
RenderBasePass() 提交绘制
```

## 🔗 核心类索引

| 类 | 职责 |
|----|------|
| `FBasePassMeshProcessor` | BasePass网格处理 |
| `FMeshDrawCommand` | 缓存的绘制命令 |
| `FMeshDrawCommandStorage` | 命令存储 |
| `FMeshBatch` | 网格批次数据 |

## 🔗 核心函数索引

| 函数 | 文件 | 作用 |
|------|------|------|
| `RenderBasePass()` | BasePassRendering.cpp | 主入口 |
| `AddMeshBatch()` | MeshPassProcessor.cpp | 添加网格 |
| `BuildMeshDrawCommands()` | MeshPassProcessor.cpp | 构建命令 |
