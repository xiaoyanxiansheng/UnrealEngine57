# 5.3 RenderDoc实践 - 速查卡

## 🎯 核心结论

1. **启动参数**：`-d3d12 -RenderDocPlugin`
2. **抓帧快捷键**：F12
3. **Event Browser**：查看所有渲染事件
4. **Texture Viewer**：查看RT内容
5. **Pipeline State**：查看绑定状态

## 📊 常用分析流程

```
1. 启动编辑器（带RenderDoc参数）
2. 打开RenderDoc，Attach进程
3. F12抓帧
4. Event Browser找到目标Pass
5. Texture Viewer查看输入输出
6. Pipeline State检查Shader绑定
```

## 🔗 关键分析点

| 要素 | 查看位置 |
|------|----------|
| GBuffer内容 | Texture Viewer → GBufferA/B/C/D |
| Shader代码 | Pipeline State → VS/PS |
| 绑定资源 | Pipeline State → Resources |
| Draw Call | Event Browser |
| 时间统计 | Event Browser → Duration |

## 💡 常用技巧

| 技巧 | 说明 |
|------|------|
| Overlay | 显示特定通道（R/G/B/A） |
| Range | 调整显示范围 |
| Pixel History | 像素历史追踪 |
| Debug | 单步调试Shader |

## 📊 RenderDoc vs Unreal Insights

| 工具 | 优势 |
|------|------|
| RenderDoc | GPU资源细节、Shader调试 |
| Insights | CPU/GPU时间线、整体性能 |
