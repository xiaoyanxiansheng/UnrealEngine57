# 5.1 控制台命令系统 - 速查卡

## 🎯 核心结论

1. **三类对象**：FAutoConsoleCommand（命令）、TAutoConsoleVariable（变量）、FAutoConsoleVariableRef（引用变量）
2. **存储结构**：IConsoleManager内部TMap<FString, IConsoleObject*>，O(1)哈希查找
3. **线程安全**：ECVF_RenderThreadSafe → Shadow Copy双份值，GameThread写[0]、RenderThread读[1]
4. **自动注册**：static声明 → 构造函数注册 → 析构函数反注册
5. **优先级**：SetByConstructor < SetByScalability < SetByConsole（控制台输入最高）

## 📊 三种注册方式

| 方式 | 用途 | 读取开销 |
|------|------|---------|
| `TAutoConsoleVariable<T>` | 内部存储变量 | ~2ns（GetValueOnRenderThread） |
| `FAutoConsoleVariableRef` | 绑定C++全局变量 | 0ns（直接读变量） |
| `FAutoConsoleCommand` | 执行一次性操作 | N/A |

## 📊 渲染CVar必备标志

```
ECVF_RenderThreadSafe    (0x20)  所有渲染CVar必须
ECVF_Scalability         (0x40)  可被画质等级控制
ECVF_Cheat               (0x01)  Shipping中隐藏
```

## 💡 快速模板

```cpp
// 注册RenderThread安全CVar
static TAutoConsoleVariable<int32> CVarMyFeature(
    TEXT("r.MyFeature"), 0,
    TEXT("0=Off, 1=On"),
    ECVF_RenderThreadSafe);

// 注册命令
static FAutoConsoleCommand CCmdMyDebug(
    TEXT("MyDebug.DoSomething"),
    TEXT("Description"),
    FConsoleCommandDelegate::CreateStatic([]() {
        UE_LOG(LogTemp, Warning, TEXT("Done!"));
    }));

// RenderThread中读取CVar
int32 Value = CVarMyFeature.GetValueOnRenderThread();
```
