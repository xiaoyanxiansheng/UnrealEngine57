# RenderingPractice 长期路线

## 当前课程

```text
创建工程
→ Rendering Room
→ Rendering Building
→ Rendering District
→ Rendering World
→ Scalability 与最终性能验收
```

课程同时增长四种能力：

- UE 编辑器、资产和工程操作
- 材质、灯光、后处理与实时渲染
- 环境搭建、模块化资产和场景组织
- 大世界、流送、内存与性能优化

## 阶段产出

| 阶段 | 产出 |
|---|---|
| Room | 可用于材质、灯光和基础分析的测试房间 |
| Building | 使用 Nanite、Lumen 和 VSM 的室内外建筑 |
| District | 使用实例化和 MegaLights 的夜景街区 |
| World | 使用 Landscape、Foliage、PCG、World Partition 和 HLOD 的开放世界 |
| Final | 具备画质档位、性能基线和技术说明的完整工程 |

## 后续案例与作品集

课程工程完成后再进入独立阶段：

```text
分析外部案例
→ 仿制关键技术
→ 比较实现路径
→ 提炼自己的环境与渲染方案
→ 形成 TA 作品集
```

当前课程产生的地图、资产、性能捕获和问题记录可直接作为后续素材，但不要求每篇都包装成作品集文档。
